# 🏥 산타간호 (Santa Nurse)

신규 간호사를 위한 **맞춤형 적응형 학습 퀴즈 앱**입니다.
산타토익처럼 진단 테스트로 실력을 가늠한 뒤, 취약한 영역을 집중 공략하는 문제를 자동으로 출제합니다.

3. `산타간호.apk` 파일을 탭하여 설치

## ✨ 주요 기능

- **실력 진단 테스트** — 10문제로 현재 수준 측정
- **적응형 출제** — 약한 영역에 가중치를 두어 맞춤 문제 자동 생성, 실력이 오르면 난이도도 상승
- **즉시 피드백 + 해설** — 정답·오답 즉시 확인, 임상 해설 제공
- **오답 복습** — 세션 종료 후 틀린 문제 전체 리뷰
- **실력 분석 대시보드** — 카테고리별 점수, 레벨(입문→전문가), 정답률, 연속 정답 스트릭
- **오프라인 동작** — localStorage 기반 학습 진도 저장

## 🛠️ 개발자용 — 로컬 실행 및 빌드

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 웹 빌드
npm run build
```

### 안드로이드 APK 다시 빌드하기

```bash
npm run build              # 웹 에셋 빌드
npx cap sync android       # 네이티브 프로젝트에 동기화
cd android
./gradlew assembleDebug    # APK 생성 (Java 21 필요)
# 결과물: android/app/build/outputs/apk/debug/app-debug.apk
```
